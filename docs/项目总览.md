# XXXX协会科学技术奖评审管理系统 - 项目总览

## 🎯 项目概述

本项目是一个完整的前后端分离的Web应用系统，专为科技奖项评审流程设计。系统实现了从申报、推荐、初审、专家评审、评审委员会终审到公示归档的全流程数字化管理。

### 主要特点

- ✅ **完整的业务流程** - 覆盖评奖全生命周期
- ✅ **多角色权限管理** - 7种不同角色，精细化权限控制
- ✅ **专家匿名评审** - 保证评审公平公正
- ✅ **数据统计分析** - 多维度数据可视化
- ✅ **文件管理** - 支持多种格式文件上传下载
- ✅ **标准RESTful API** - 前后端完全分离
- ✅ **完善的文档** - 包含部署、API、数据库设计文档

## 👥 系统角色

系统支持7种用户角色:

1. **系统管理员 (admin)**
   - 完全控制权限
   - 用户管理、系统配置
   - 数据维护

2. **协会工作人员 (staff)**
   - 受理申报
   - 初审材料
   - 组织评审流程
   - 发布公示

3. **推荐单位管理员 (recommender)**
   - 推荐申报项目
   - 上传推荐材料

4. **申报单位 (applicant)**
   - 在线申报
   - 上传材料
   - 查看进度

5. **专家评审 (expert)**
   - 匿名评审
   - 评分打分
   - 提交评审意见

6. **评审委员会 (committee)**
   - 终审决议
   - 投票表决
   - 确定奖项等级

7. **公众 (public)**
   - 查看公示
   - 提交异议

## 🔄 业务流程

```
1. 申报阶段
   ├── 申报人创建申报 (draft)
   ├── 完善申报信息
   ├── 上传材料附件
   └── 提交申报 (submitted)

2. 推荐阶段
   ├── 推荐单位审核
   ├── 上传推荐文件
   └── 确认推荐 (recommended)

3. 初审阶段
   ├── 协会工作人员审核
   ├── 材料完整性检查
   └── 初审结果 (preliminary_approved/rejected)

4. 专家评审阶段
   ├── 分配专家 (expert_review)
   ├── 专家评分
   ├── 提交评审意见
   └── 汇总评分

5. 评审委员会阶段
   ├── 评委会审议 (committee_review)
   ├── 投票表决
   ├── 确定奖项等级
   └── 终审结果 (approved/rejected)

6. 公示阶段
   ├── 发布公示 (announced)
   ├── 接受异议
   ├── 处理异议
   └── 公示结束

7. 归档阶段
   └── 存档备案 (archived)
```

## 📝 API文档

系统提供完整的Swagger API文档，启动后访问:

- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

主要API端点:

```
POST   /api/auth/login              # 用户登录
GET    /api/auth/me                 # 获取当前用户信息
GET    /api/users/                  # 用户列表
POST   /api/users/                  # 创建用户
GET    /api/organizations/          # 组织列表
GET    /api/awards/                 # 奖项列表
GET    /api/applications/           # 申报列表
POST   /api/applications/           # 创建申报
POST   /api/applications/{id}/submit # 提交申报
GET    /api/reviews/my-reviews      # 我的评审任务
POST   /api/reviews/                # 创建评审
GET    /api/statistics/overview     # 统计概览
```

## 🚀 快速开始

### 1. 环境准备
```bash
# 安装Python 3.8+, Node.js 16+, MySQL 5.7+
```

### 2. 后端启动
```bash

python setup_database.py
```
